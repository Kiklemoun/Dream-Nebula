# 🌌 梦幻星云 (Dream Nebula) - CV 手势交互 3D 特效

## 📖 项目简介
本项目是一个基于原生 Web 技术与计算机视觉构建的 3D 粒子互动艺术作品。
它内置了 Google MediaPipe Hands 视觉引擎，**无需配置 Python 环境，无需下载任何权重文件**。只需一个带有摄像头的浏览器，即可通过真实手势，体验“修仙施法”般的宇宙星云操控感。

## ✨ 核心特性
* **零依赖，开箱即用**：纯 HTML/JS 实现，双击即可运行。
* **高鲁棒性手势检测**：重构了基于“手腕节点”的尺度不变性算法，无论摄像头是平视还是高角度俯视，均能精准识别。
* **电影级物理渲染**：内置 20000 颗恒星粒子与 5000 颗星云粒子，包含柔和引力、自然回弹与呼吸闪烁的星星雨特效。

## 🕹️ 手势交互指南 (操作说明)
请允许浏览器访问摄像头，并站在距离摄像头适中的位置。系统支持以下 4 种硬核交互：

1. **💥 终极爆发 (剑诀 -> 全开)**
   * **前置动作**：摆出“剑诀”（仅伸直食指和中指，其余弯曲），系统提示“剑诀已成”。
   * **爆发动作**：手掌猛然完全张开！算法会捕捉你手指张开的瞬间加速度，产生巨大的宇宙冲击波，将星云炸成满天星雨。
2. **🌀 极速聚气 (握拳)**
   * 只要握紧拳头，散开的星云会受到高达 25 倍的引力，瞬间坍缩聚合成高密度恒星。
3. **✨ 缓慢汇聚 (五指并拢)**
   * 五指伸直但指尖紧紧靠拢（手掌变窄），星云会以平缓的姿态优雅地汇聚。
4. **🪐 星云流转 (五指张开 + 左右平移)**
   * 在五指自然张开的状态下，在空中左右平移手掌，星云会跟随着你的手势，以绝美的侧俯视倾角进行 3D 旋转。
5. **🔒 锁定形态 (蜘蛛侠手势)**
   * 伸出大拇指、食指和小拇指，星云将被死死锁定在当前状态，此时放下手也不会散开。
